# new-flask-app
> Create a new flask application using this simple command

## Install
> To install it on your system:

    $ python setup.py install

## Usage
> To create a new flask application run:

    $ new-flask-app

> This will create a directory called `app` with the basic
> flask structure:


    └── app
        ├── APP_NAME
        │   ├── __init__.py
        │   ├── __init__.pyc
        │   ├── app.py
        │   ├── app.pyc
        │   ├── static
        │   │   ├── css
        │   │   ├── image
        │   │   └── js
        │   │       ├── app.js
        │   │       ├── packed.js
        │   │       └── utils.js
        │   ├── templates
        │   │   ├── index.html
        │   │   └── layout.html
        │   └── views
        │       ├── __init__.py
        │       ├── __init__.pyc
        │       ├── index.py
        │       └── index.pyc
        ├── __main__.py
        └── setup.py
